# **OTA Programmer**
****
## **Project Description**
    The aim of this project to load new images wirelessly to a specific host using NodeMCU for:
            1. Load the image.
            2. Check the image integrity.
            3. Burn the image to the host.

    Using TM4C123 as a host to test our wireless burner.
